# Canvas Grade Calculator Extension
Student-focused Chrome extension that analyzes Canvas assignments to calculate weighted grades and predict course outcomes.

## Features
- Extracts assignment data directly from Canvas pages
- Calculates current weighted grade automatically
- Allows “what-if” scenarios for remaining assignments
- Displays results in a simple popup dashboard

## Motivation
Students often struggle to quickly understand their standing in a course or predict their final grade (including me).  
This tool provides instant grade insights without needing manual spreadsheets.

## Tech Stack
- JavaScript (DOM parsing and calculations)
- HTML/CSS (popup interface)
- Chrome Extension Manifest V3
